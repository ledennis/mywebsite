Using Links

Surround a text with <a></a> to create a link, a stands for anchor
using href inside the first bracket, you can set the link to href="somewebsite"
e.g. <a href="http://google.com"> will create a link to google.com
You can link to your own pages in your directory as well, link them by doing
<a href="newpage.html"> which will link to this new page

How to make a table of contents

Making a table of contents is similar to creating links but instead of referrencing to links, you referrence to certain parts of your page by using name tags. To create a name tag, you must put a <a name="Whatever name tag you want"><a/> in a section of your page where you want the jump to occur. To link the "jump", surround a text with <a href="#name tag">text used for link</a>

e.g. I created the name tag "toc1" in the header for Table of Contents in the beggining of the page. <h2><a name="toc1"><a/>Table of Contents</h2>.
This link will go to the table of contents. <a href=#"toc1">link</a>
