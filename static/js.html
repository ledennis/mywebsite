To create the adding feauture, an input was needed to get the user input and then event listeners were added to know whenever a number was entered. Listeners can wait for these kind of events:

Events

  • mouseenter: when mouse enters
  • mouseleave: when mouse leaves

  • mouseup: when mouse moves up

  • keydown: when down key is entered

  • blur: after something is left

The HTML code is pretty straightforwad, there are two input elements that take in the user input and they are ID'd accordingly so that they can be found and used in the Javascript file.

Javascript Code for Add Program


var numOne = document.getElementById("num-one");
var numTwo = document.getElementById("num-two");
var addSum = document.getElementById("add-sum");

numOne.addEventListener("blur", add);
numTwo.addEventListener("blur", add);

function add() {
  var one = parseFloat(numOne.value) || 0;
  var two = parseFloat(numTwo.value) || 0;
  addSum.innerHTML = "Your sum is " + (one + two);
}

For the Javascript portion, we used selectors to find the input elements in the HTML and stored them in two variables. Afterwords two even listeners are created that are triggered by the "blur" event, which adds after the user clicks ouside of the input field. The add function uses the variables that have the inputs stored and converts it into a Float value using parseFloat(). The or operator is used so that if a number is not used in the input field, 0 will be added instead. We can change the HTML code of the addSum by changing it's innerHTML, which is how the added values are shown.

Javascript Code for List Editor


var checklist = document.getElementById("checklist");

var items = checklist.querySelectorAll("li");
var inputs = checklist.querySelectorAll("input");


for (var i = 0; i < items.length; i++) {
  items[i].addEventListener("click", editItem);
  inputs[i].addEventListener("blur", updateItem);
  inputs[i].addEventListener("keypress", itemKeyPress);
}

function editItem() {
  this.className = "edit";
  var input = this.querySelector("input");
  input.focus();
  input.setSelectionRange(0, input.va)
}

function updateItem() {
  this.previousElementSibling.innerHTML = this.value;
  this.parentNode.className = "";
}

function itemKeyPress(event) {
  if (event.which === 13) {
    updateItem.call(this);
  }
}

There are three listening events, one for when an item in the list is clicked, one for when the mouse leaves the input field after, and a keypress event for the enter key. When an item is clicked, the input field replaces the text and afterwards the text is updated after a value is entered or clicked out of.
